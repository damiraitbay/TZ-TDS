# Приложение для управления пользователями

Веб-приложение для управления списком пользователей, созданное с использованием React.js, TypeScript, React Router и React Hook Form. Приложение интегрировано с реальным REST API для работы с данными пользователей.

## Функциональность

- ✅ Просмотр списка пользователей в виде таблицы
- ✅ Добавление нового пользователя
- ✅ Редактирование существующего пользователя
- ✅ Удаление пользователя
- ✅ Пагинация списка пользователей
- ✅ Сортировка по различным полям (ID, Имя, Фамилия, Email, Дата регистрации)
- ✅ Валидация форм (обязательные поля, формат email)
- ✅ Управление навыками (добавление/удаление)
- ✅ Использование React Query для управления состоянием
- ✅ Интеграция с реальным REST API

## Технологии

- **React 18** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **React Router** - маршрутизация в React приложении
- **React Hook Form** - библиотека для работы с формами
- **React Query (TanStack Query)** - управление серверным состоянием и кэширование
- **Tailwind CSS** - утилитарный CSS фреймворк
- **Vite** - инструмент сборки
- **date-fns** - библиотека для работы с датами

## API

Приложение использует REST API для работы с данными пользователей:

**Base URL:** `https://69668b8df6de16bde44db645.mockapi.io/users`

### Эндпоинты API

- `GET /users` - получение списка всех пользователей
- `GET /users/:id` - получение пользователя по ID
- `POST /users` - создание нового пользователя
- `PUT /users/:id` - обновление пользователя
- `DELETE /users/:id` - удаление пользователя

### Структура данных пользователя

```typescript
interface User {
  id: number;
  firstName: string;
  lastName: string;
  email: string;
  skills: string[];
  registrationDate: string; // ISO формат даты
}
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение в режиме разработки:
```bash
npm run dev
```

3. Откройте браузер и перейдите по адресу, указанному в терминале (обычно `http://localhost:5173`)

4. Для сборки production версии:
```bash
npm run build
```

5. Для предпросмотра production сборки:
```bash
npm run preview
```

## Особенности реализации

### Работа с API

Приложение полностью интегрировано с REST API. Все операции с данными (создание, чтение, обновление, удаление) выполняются через HTTP-запросы. API клиент (`src/services/api.ts`) обрабатывает:

- Преобразование данных из формата API в формат приложения
- Обработку ошибок и статус-кодов HTTP
- Поддержку различных форматов данных (строковые/числовые ID, createdAt/registrationDate)

### Валидация форм

Формы валидируются на клиентской стороне с использованием React Hook Form:

- **Имя и Фамилия**: обязательные поля, минимум 2 символа
- **Email**: обязательное поле, проверка корректного формата email
- **Навыки**: опциональное поле, можно динамически добавлять/удалять элементы

### Пагинация

Список пользователей разбит на страницы по 10 элементов. Внизу таблицы отображается навигация по страницам с информацией о текущей странице и общем количестве пользователей.

### Сортировка

Таблица поддерживает сортировку по следующим полям:
- ID (числовая сортировка)
- Имя (алфавитная сортировка)
- Фамилия (алфавитная сортировка)
- Email (алфавитная сортировка)
- Дата регистрации (сортировка по дате)

Клик по заголовку столбца переключает направление сортировки (по возрастанию/убыванию). Визуальный индикатор показывает текущее поле и направление сортировки.

### Управление состоянием

Приложение использует React Query (TanStack Query) для:
- Кэширования данных пользователей
- Автоматической синхронизации состояния после мутаций
- Оптимистичных обновлений UI
- Обработки состояний загрузки и ошибок
